# IBKR

Fetch and analyze US stock data using tickers from a CSV.

## Setup

```bash
python -m venv venv
source venv/bin/activate   # or: venv\Scripts\activate on Windows
pip install pandas yfinance
```

## Usage

- **Fetch 3 years OHLC + earnings** for all tickers in `tickers.csv` (writes one CSV per ticker to `data/`):

  ```bash
  python fetch_ohlc_earnings.py
  ```

  Options: `--limit N` (test on N tickers), `--no-resume` (re-fetch all), `--verbose` (yfinance debug).

- Re-run to resume; rate limits are handled with wait + retry.

## Data

- `tickers.csv` — input list (ticker, name, industry, marketCap).
- `data/<TICKER>.csv` — daily OHLC, volume, and earnings-day columns (generated by the script; not committed).
